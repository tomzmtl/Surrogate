# Surrogate

Surrogate is an API that delivers random dummy content to web applications.

## Endpoints

### Images

#### api/img/src

Random image URL.

#### api/img/tag (todo)

HTML code of a random image.

### Paragraphs

#### api/lorem/{N} (todo)

N paragraphs of dummy text.
